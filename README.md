# 🚀 DevFolio – Personal Portfolio Website

Welcome to **DevFolio**, a modern and responsive personal portfolio website built to showcase professional skills and projects.

This project was developed as part of **Assignment 7**, focusing on creating a smooth user experience using advanced CSS techniques, Bootstrap 5, and interactive JavaScript features.

## ✨ Features

### 📱 Responsive Architecture
Built using a **Mobile-First approach** so the website adapts beautifully to phones, tablets, and desktops.

### ⌨️ Dynamic Typed Text
Integrated **Typed.js** to display multiple professional roles with a typing animation.

### 🧭 Smart Navigation Bar
Navbar changes its style dynamically when the user scrolls.

### 📊 Performance Counters
Animated statistics section highlighting achievements and project numbers.

### 🖼️ Portfolio Gallery
A project showcase with hover effects for better user interaction.

### ⬆️ Scroll to Top Button
A smooth **Back to Top** button that appears when the user scrolls down.

## 💡 Customization

To change the animated text in the hero section, edit the script at the bottom of **index.html**:

```javascript
var typed = new Typed('.auto-type', {
  strings: ["Your Name", "Web Developer", "Graphic Designer"],
  typeSpeed: 70,
  backSpeed: 70,
  loop: true
});
You can replace the roles with your own professional titles..
